# Discord Bidding Bot

Automated bidding system for Discord threads.

## Features
- Structured bidding
- Phase-based system (24h open → restricted → auto close)
- Auto %pay generation
- Invalid bid prevention
- Leader tools for moderation

## Setup

1. Install dependencies:
pip install -r requirements.txt

2. Set environment variable:
DISCORD_TOKEN=your_token_here

3. Run bot:
python bot.py
